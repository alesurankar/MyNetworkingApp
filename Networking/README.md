
# Networking (Static Library)

This project is a **static library** that provides TCP networking functionality
based on Boost.Asio.

It is used by Client and Server.


## Project Structure

```
Networking/
├─ include/
│	└─ networking/
│		├─ connection.hpp
│		└─ message.hpp
│
└─ src/
	└─ connection.cpp
```
